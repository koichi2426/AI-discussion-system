# AI-discussion-system
## 概要
自然言語モデルが今流行りのニュースについて議論して、結局我々は何ができるのかを提案してくれるyoutubeチャンネル。

## 開発背景
理由１:皆さん普段ニュースを見ると思いますが、ただ見るだけにとどまっていないでしょうか。そこで私はそのニュースから個人ができること、国ができることを提案してもらうyoutubeチャンネルを開発しました。

理由２:動画編集と投稿の全自動化ってロマンがありますよね。

## システム
① 今流行っているニュースを取得する。

② 取得したニュースについてChatGPTやClaudeなどの自然言語処理AIに議論をしてもらい、その問題を解決するために国ができること、我々個人ができることを結論付けてもらう。

③ 以上の内容を3分程度の動画に編集する。

③ ①~③を毎日12時に自動実行しyoutubeに動画を投稿する。
